# [연습문제] 문자열 다루기 기본

* 2020/08/06

```java
class Solution {
    public boolean solution(String s) {
        boolean answer = true;
        
        if(s.length() != 4 && s.length() != 6) {
            return false;
        } else {
            for(int i = 0; i < s.length(); i++) {
                if(!(s.charAt(i) >= '0' && s.charAt(i) <= '9')) {
                    answer = false;
                }
            } 
        }
        
        
        return answer;
    }
}
```

- '0' ~ '9' : 숫자의 문자코드
