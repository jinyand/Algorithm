# [연습문제] 가운데 글자 가져오기

* 2020/08/06

```java
class Solution {
    public String solution(String s) {
        String answer = "";
        int length = s.length();
            
        if(length % 2 == 0) {
            answer = s.substring(length/2 - 1, length/2 + 1);
        } else {
            answer = s.substring(length/2, length/2 + 1);
        }
        
        return answer;
    }
}
```

- substring(a, b) 라면 a부터 b-1까지 출력한다는 걸 생각하기
→ 그래서 substring의 두번째 인자에 + 1 을 해주는 것
