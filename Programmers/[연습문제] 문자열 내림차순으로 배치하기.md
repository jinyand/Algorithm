# [연습문제] 문자열 내림차순으로 배치하기

* 2020/08/13

```java
import java.util.*;

class Solution {
    public String solution(String s) {
        String answer = "";
        
        char[] str = s.toCharArray();
        Arrays.sort(str);
        
        StringBuilder sb = new StringBuilder(new String(str));
        answer = sb.reverse().toString();
        
        return answer;
    }
}
```

- StringBuilder의 reverse와 toString을 활용하여 역순으로 배치했다.
