# [연습문제] 2016년

* 2020/08/11

```java
class Solution {
    public String solution(int a, int b) {
        String answer = "";
        int[] month = {31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
        int day = 0;
        
        for(int i = 0; i < a - 1; i++) {
            day += month[i];
        }
        
        day += b - 1;
        
        switch(day % 7) {
            case 0:
                answer = "FRI";
                break;
            case 1:
                answer = "SAT";
                break;
            case 2:
                answer = "SUN";
                break;
            case 3:
                answer = "MON";
                break;
            case 4:
                answer = "TUE";
                break;
            case 5:
                answer = "WED";
                break;
            case 6:
                answer = "THU";
                break;
        }
        
        return answer;
    }
}
```

- a월 b일 의 날짜를 구한다고 했을 때
- 먼저 2016년 이후 ~ b일 까지의 일 수를 계산한다.
→ ex) 2월 5일이면 31일(1월) + 5일(2월) 에서 1월 1일 (1일)을 빼준다.
- 여기서 7을 나눈 나머지를 통해 0 : 금요일, 1 : 토요일 ...
