# [연습문제] 하샤드 수

* 2020/08/13

```java
import java.util.*;

class Solution {
    public boolean solution(int x) {
        boolean answer = true;
        List<Integer> list = new ArrayList<Integer>();
        int a = x;
        int sum = 0; 
        
        while(a > 0) {
            list.add(a % 10);
            a /= 10;
        }
        
        for(int i = 0; i < list.size(); i++) {
            sum += list.get(i);
        }
        
        if(x % sum == 0) answer = true;
        else answer = false;
        
        return answer;
    }
}
```

- x를 복사한 a라는 변수를 사용해주어야 한다는 점
